## 📖 Sobre o Projeto

O **Bee Better** é uma aplicação mobile desenvolvida para auxiliar pessoas com **TDAH (Transtorno do Déficit de Atenção e Hiperatividade)** a organizarem suas rotinas de forma mais intuitiva, visual e motivadora.

Muitas ferramentas tradicionais de produtividade dependem exclusivamente de disciplina e organização. Para pessoas neurodivergentes, isso pode gerar frustração e abandono da ferramenta.

Pensando nisso, o Bee Better utiliza conceitos de:

- 🎮 Gamificação
- 🧠 Psicologia Cognitiva
- 📈 Reforço Positivo
- 📅 Organização de Rotina
- 🎯 Gestão de Metas

para transformar tarefas cotidianas em uma experiência mais engajadora.

---

## 🧠 O Problema

Pessoas com TDAH frequentemente enfrentam dificuldades relacionadas a:

- Iniciar tarefas;
- Manter o foco;
- Organizar prioridades;
- Gerenciar o tempo;
- Lembrar compromissos;
- Manter consistência em hábitos.

O Bee Better busca minimizar esses desafios através de mecanismos visuais e recompensas constantes.

## ⚙️ Como Executar

### 1. Clone o repositório

```bash
git clone https://github.com/Matth998/bee-better.git
```

### 2. Entre na pasta do projeto

```bash
cd bee-better
```

### 3. Instale as dependências

```bash
flutter pub get
```

### 4. Execute a aplicação

```bash
flutter run
```

## 🔐 Variáveis de Ambiente

Para execução local do backend, configure as variáveis de ambiente necessárias:

```env
JWT_SECRET=
DATABASE_USER=
DATABASE_PASSWORD=
```

As credenciais reais não são armazenadas no repositório por questões de segurança.
